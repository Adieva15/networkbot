# Нейросетевой помощник — веб-приложение

Веб-приложение на Flask, предоставляющее доступ к 10+ нейросетевым функциям: анализ тональности, генерация текста, суммаризация, распознавание объектов, удаление фона, раскрашивание фото, определение эмоций, перенос стиля, создание мемов и история (в процессе разработки). Модели загружаются локально с Hugging Face (через зеркало `hf-mirror.com`).

## Возможности

| Функция | Описание | Тип ввода |
|---------|----------|-----------|
| **Анализ тональности** | Определяет эмоциональную окраску текста (негатив/нейтраль/позитив) | Текст |
| **Генерация текста** | Продолжает ваш промпт на основе GPT‑2 | Текст |
| **Суммаризация** | Сжимает длинный текст в краткий пересказ (T5/mT5) | Текст |
| Распознавание объектов | Находит объекты на фото (DETR) | Изображение |
| Удаление фона | Убирает фон с портрета или предмета (Replicate) | Изображение |
| Раскрашивание | Превращает ч/б фото в цветное | Изображение |
| Определение эмоций | Распознаёт эмоцию по лицу на фото | Изображение |
| Перенос стиля | Перерисовывает одно фото в стиле другого (Replicate) | Два изображения |
| Создание мема | Накладывает ваш текст на шаблон с картинкой | Текст |

## Установка и запуск

### Требования
- Python 3.10+
- Оперативная память: от 4 ГБ (модели загружаются в ОЗУ)
- Диск: около 2–3 ГБ для кеша моделей

### Шаги

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/neural-web.git
   cd neural-web
   ```

2. Создайте виртуальное окружение:
   ```bash
   python -m venv .venv
   source .venv/bin/activate      # Linux/Mac
   .venv\Scripts\activate          # Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Настройте `config.py` (при необходимости измените модели). По умолчанию используются зеркало Hugging Face и следующие модели:
   - Тональность: `nlptown/bert-base-multilingual-uncased-sentiment`
   - Генерация: `gpt2`
   - Суммаризация: `IlyaGusev/rut5_base_sum_gazeta`
   - Распознавание объектов: ``
   - Эмоции: ``
   - (Для удаления фона и стиля требуется ключ Replicate — опционально)

5. Запустите приложение:
   ```bash
   python app.py
   ```

6. Откройте браузер по адресу:  
   `http://127.0.0.1:5000`

> **При первом запуске** модели будут загружены с `hf-mirror.com`. Это может занять несколько минут (размер файлов от 300 МБ до 1.5 ГБ). Прогресс отображается в консоли.

## Использование

### Через веб-интерфейс

1. Выберите нужную функцию из выпадающего списка.
2. Введите текст или загрузите изображение(я).
3. Нажмите «Отправить».
4. Результат появится на той же странице (текст или изображение).

### Примеры запросов

- **Тональность**:  
  *«Фильм был потрясающим!»* → `Позитивный`
- **Генерация**:  
  *«Однажды в студёную зимнюю пору…»* → продолжение текста
- **Суммаризация**:  
  (длинная статья) → краткий абзац
- **Перенос стиля**:  
  загрузите фото содержимого (например, ваш портрет) и фото стиля (например, картину Ван Гога)

## Конфигурация

Файл `config.py` содержит переменные окружения и пути к моделям. Основные параметры:

| Переменная | Описание | Значение по умолчанию |
|------------|----------|-----------------------|
| `HF_ENDPOINT` | Зеркало Hugging Face | `https://hf-mirror.com` |
| `MODEL_NAME` | Модель для тональности | `nlptown/bert-base-multilingual-uncased-sentiment` |
| `model_gt` | Модель для генерации текста | `gpt2` |
| `model_sum` | Модель для суммаризации | `t5-small` |
| `REPLICATE_API_TOKEN` | Токен для Replicate (удаление фона, стиль) | (необязательно) |

Для использования функций удаления фона и переноса стиля необходимо получить бесплатный API‑ключ на [replicate.com](https://replicate.com) и указать его в `.env` или прямо в `config.py`. Без ключа эти две кнопки будут выдавать ошибку.

## Примечания

- Все модели работают локально, данные пользователей не покидают ваш компьютер.
- При использовании Replicate (удаление фона, стиль) изображения передаются на внешний сервер — ознакомьтесь с политикой конфиденциальности сервиса.
- Для ускорения загрузки моделей можно использовать `torch.device('cuda')`, если у вас есть GPU (требуется CUDA-версия PyTorch).
